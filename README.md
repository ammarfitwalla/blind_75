# blind_75 — LeetCode Blind 75 Solutions

A curated collection of LeetCode "Blind 75" problem solutions, organized by topic (arrays, dynamic programming, graphs, trees, etc.). Each solution is implemented in Python and placed in a topic-specific folder.

## Contents
- arrays/
- DP/
- graphs/
- heap/
- linked_list/
- matrix/
- string/
- tree/

## Usage
1. Clone the repository:

```bash
git clone https://github.com/ammarfitwalla/blind_75.git
cd blind_75
```

2. Run a solution file using Python (3.8+ recommended):

```bash
python blind_75/arrays/easy/1_two_sum.py
```
